title: Introduction
description: What is the Teachable API wrapper and why does it exist?
---

The Teachable API wrapper is a lightweight, strictly-typed Node.js SDK
for the Teachable v1 API. Built on [Zod](https://zod.dev) for runtime
validation, with full TypeScript support out of the box.
